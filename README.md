# 基于角色的权限访问控制管理项目（RBAC）

框架基于.NET8.0，使用RazorPage构建的后台管理，实现了页面、按钮（接口）权限拦截。整体项目技术简单，非常适合作为初学项目和升级到企业后台，并且有自定义无权限页面、禁止访问错误页面、动态定时任务、内存缓存key管理等特色功能。
